# Markdown

## What is Markdown?

Markdown is a lightweight markup language used to format plain text using simple symbols. It is widely used for README files, notes, blogs, and technical documentation.

## Why is it used?

- Documentation
- README file
- Blogging
- Note-taking
- Web Publishing

## Basic syntax

| Formatting | Markdown Syntax |
| Heading | ## Heading |
| Bold Text | **bold** |
| Italic Text | _italic_ |
| Link | [Name](url) |
| List (Unordered) | - Item <br> - Item 2 |
| Blockquote | > Text here |

## Advantages

- Simple to use
- Better indentation
- Portable
- Efficient
- Easy to understand the text

## Where is Markdown Used?

- GitHub README files
- Software documentation
- Technical blogs
- Note-taking applications
- Project documentation

## Summary

- Markdown is used to format plain text
- used for documentation, README files, web docs
- Easy to use, portable and efficient
