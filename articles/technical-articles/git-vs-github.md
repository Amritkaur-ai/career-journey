# Git vs Github

## What is Git?

Git is a free and open-source version control software used to track the changes in file, collaborate with other developers, and manage projects of anysize.

## What is GitHub?

GitHub is a cloud-based platform that holds repositories online. It allows developers to stores code, collaborate on projects, review changes, and manage software developments.

## Difference between Git and GitHub?

- Git is a software tool whereas GitHub is a web-based hosting service.
- Git tracks changes locally whereas GitHub shares, collaborates and manages Git repositories online.
- Git runs on computer and GitHub runs on Cloud.
- Git works offline whereas GitHub requires internet to sync repositories
- Git is open,source which is maintained by community whereas GitHub is owned by Microsoft
- Git is used for branching, commiting and maintaining history whereas GitHub shares, collobrates and manages Git repositories online.

## Why developers use both?

Git and GitHub complement each other. Git helps developers track changes locally, while GitHub makes it easy to back up repositories, collobrate with others, and share code online.
